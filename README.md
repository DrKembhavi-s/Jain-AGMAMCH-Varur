## 🔄 Monthly Update Procedure (Complete by 10th of Every Month)

### Step 1 — Collect PDFs from Departments
Collect the following PDFs from the relevant departments:

| Section | What to Collect |
|---|---|
| Teaching Staff | Updated staff list signed by Principal |
| Non-Teaching Staff | Updated staff list (college + hospital) |
| Hospital Statistics | Monthly stats report from MS office |
| Academic Activities | Reports of any CME / workshop held |
| Events | Flyers / notifications of upcoming events |
| Research | Any new publications (add to annual list) |
| PG Departments | Any new documents from HODs |

### Step 2 — Upload PDFs to GitHub

1. Go to **github.com** → open your repository
2. Navigate to the correct folder (e.g., `docs/hstats/`)
3. Click **Add file → Upload files**
4. Drag and drop your PDF → click **Commit changes**

> 💡 You can upload multiple PDFs at once by selecting all files before dragging.

### Step 3 — Add a Row in index.html (for all sections except PG)

1. In your repository, click `index.html`
2. Click the ✏️ **Edit (pencil)** icon at top right
3. Find the relevant section's document table
4. Copy an existing `<tr>...</tr>` row and paste it below
5. Update: the serial number, document name, period, date, and the PDF file path in `href="..."`
6. Click **Commit changes**

**Example — adding a new Hospital Stats row:**
```html
<!-- Copy this pattern and update the values -->
<tr>
  <td>3</td>
  <td>Hospital Statistics — Monthly Report</td>
  <td>May 2025</td>
  <td>10 Jun 2025</td>
  <td><a href="docs/hstats/stats-may-2025.pdf" class="vbtn">📄 Open PDF</a></td>
</tr>
```

### Step 4 — Add PG Documents (No HTML editing needed)

For **Post Graduate Studies**, no HTML editing is required:
1. Upload the PDF to the correct folder in GitHub (e.g., `docs/pg/kaya/`)
2. Open the live website → go to **Post Graduate** tab
3. Select the department sub-tab
4. Type the document name and paste the file path
5. Click **+ Add** — it saves automatically

### Step 5 — Verify Before Deadline

- Open the live website URL
- Click every new PDF link — confirm files open correctly
- Check that the date shown is correct
- Deadline: **10th of the month, without fail**

## 📝 File Naming Convention

Always use: **lowercase letters, numbers, and hyphens only. No spaces. No special characters.**

| Section | Format | Example |
